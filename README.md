## 🔴 Overview

**F1 Portfolio** is a visually striking personal portfolio website themed around the world of **Formula 1 racing**. Built with **Next.js 16**, **React 19**, and **Tailwind CSS v4**, it delivers a blazing-fast, fully responsive experience with an immersive motorsport aesthetic.

Every pixel of this project is engineered with the same attention to detail as an F1 car — where performance, design, and precision all work as one.

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- **Node.js** `v18+`
- **npm** `v9+` or **yarn**

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/I-SHOW-AKIRU200/F1-PORTFOLIO.git

# 2. Navigate into the project directory
cd F1-PORTFOLIO

# 3. Install all dependencies
npm install

# 4. Fire up the development server 🏎️
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser — **go go go!**
